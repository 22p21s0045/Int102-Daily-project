keyframe
หลักๆจะใช้ในการทํา animation ก็จะมี attribute ให้ใช้หลักๆอย่าง from , to เพื่อที่กําหนดการเคลื่อนไหวของ element
object fit
ก็จะใช้คู่กับ tag img ใช้เพื่อให้ปรับขนาดรูปให้เท่ากับขนาด element เเบบพอดีไม่ต้องปรับเอง
z index
ใช้ในการกําหนดเหมือนทํา Layer ว่าให้ element ไหนขึ้นก่อน จะใช้เป็นตัวเลข ลบในการอ้างอิง layer ยิงติดลบมากก็ยิ่งไปอยู่ข้างหลัง
IntersectionObserver function
เป็น built in function ของ Javascript ใช้ดูว่าตอนนี้ user เห็นelement ที่ต้องการอยู่มัย
animation
ใช้ในการกําหนด animation ทั้งหมดไม่ว่าจะเป็น ระยะเวลา keyframe
Svg file
ตอนเเรกเวลาขยายภาพเเล้วมันเเตกก็เลยไป เจอ svg file พอใช้เเล้วภาพไม่มีเเตกเลย ใช้กับ tag img ได้ปกติ
animation-play-state
ใช้ในการกําหนดการเล่น animation เช่น สมมุติพอ hover ให้หยุด animation
lottie file
ตอนเเรกใช้ gif ในการทําภาพเคลื่อนไหวเเต่ภาพมันเเตกก็เลยไปหาวิธีก็ไปเจอ lottie file ใส่ในมันจะเป็น ไฟล์ json ก็เลยทําให้นํ้าหนักเบาด้วยวิธีใช้ก็จําเป็นต้อง import ตัวเล่นมาก่อนหลักการของมันก็ตือจะเอาข้อมูลใน json มา render เป็นไฟล์ svgเคลื่อนไหว
